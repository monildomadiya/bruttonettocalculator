# 🇩🇪 Brutto-Netto Rechner (Germany)

A premium, high-performance Salary Calculator (Brutto-Netto-Rechner) built with Next.js 15, optimized for the German market.

## 🚀 Features

- **Precision Calculations**: Accurate tax and social security deductions for the current tax year.
- **Dynamic UI**: Real-time updates with zero input latency.
- **SEO Optimized**: Pre-rendered pages with automated metadata and schema injection.
- **Responsive Design**: Premium mobile-first experience using a flat-design system.

## 🛠️ Tech Stack

- **Framework**: [Next.js 15+](https://nextjs.org/)
- **Logic**: Custom calculation engine for German tax laws.
- **Styling**: Vanilla CSS / Tailwind (Modern Flat Design).
- **Deployment**: Optimized for Vercel or Firebase Hosting.

## 🏃 Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## 📁 Project Structure

- `src/app`: Next.js App Router pages and layouts.
- `src/components`: Reusable UI components.
- `scripts/`: Utility scripts for SEO and layout generation.
- `public/`: Static assets and SEO files.
